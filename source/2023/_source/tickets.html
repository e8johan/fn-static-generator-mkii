Get your ticket via the ticket registration page (requires the setup of an account).

The ticket registration process requires you to register using an OpenID source. At the moment we support github and Google. If you cannot identify using these services we offer a manual process. Please notice that this process involves a substantial amount of additional work for the organizers so we urge you to consider using the standard process which is quicker and easier. To request a ticket manually, please email us at info@foss-north.se with the subject line "Ticket foss-north 2023".
